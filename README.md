# 🌿 Del Salado

Sitio web oficial de **Del Salado** — productos naturales de nuestra finca familiar en San Miguel, Santander, Colombia.

![Del Salado](img/tostando-cafe.jpg)

## 🛒 Productos

- 🍯 **Miel de abejas** — Tarro 500cc
- 🌼 **Polen natural** — Bolsa 250gr
- ☕ **Café del Salado** — Libra / Media libra
- 🥑 **Aguacate Hass** — Por libra
- 🥚 **Huevos de codorniz** — Caja de 24 unidades
- 🎁 **Combo Despertar** — Café + Miel

## 🚀 Demo

Sitio en vivo: [https://TU-USUARIO.github.io/delsalado](https://TU-USUARIO.github.io/delsalado)

## 📁 Estructura

```
delsalado/
├── index.html       # Sitio principal
├── img/             # Imágenes optimizadas
├── ADMIN.md         # Guía de administración
└── README.md        # Este archivo
```

## ✏️ Administración

Todos los precios, productos y contactos se editan en un solo lugar: busca `const CONFIG` dentro de `index.html`. Ver `ADMIN.md` para la guía completa.

## 🛠️ Tecnología

- HTML5 + CSS3 + JavaScript vanilla
- Sin dependencias externas (excepto Google Fonts)
- Mobile-first responsive design
- Imágenes optimizadas para web
